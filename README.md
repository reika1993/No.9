レスポンスボディは次のとおりとする。

```bash
{
    "error": "Not Found",
    "timestamp": "2024-01-09T21:10:38.070219+09:00[Asia/Tokyo]",
    "message": "player not found",
    "status": "404",
    "path": "/players/500"
}
```
