# 概要

第9回目課題：Controller、Service、Mapperを利用してRead処理を作成後、例外をハンドリングする。

レスポンスボディは次のとおりとする。

```bash
{
    "error": "Not Found",
    "timestamp": "2024-01-09T21:10:38.070219+09:00[Asia/Tokyo]",
    "message": "player not found",
    "status": "404",
    "path": "/players/500"
}
```

## 手順

[第8回課題](https://github.com/reika1993/No.8)を利用するため、ハンズオンする。
